# internship-15-elevate-labs
Vulnerability Assessment &amp; Risk Prioritization
This project demonstrates how to perform a Vulnerability Assessment and Risk Prioritization using Nessus Essentials (primary tool) and OpenVAS (alternative).
The goal is to identify vulnerabilities, assess their severity using CVE & CVSS, classify risks, and prioritize remediation.
